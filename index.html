Games

SDL Snake

It's snake! Made in C! Using SDL! Programmed in one weekend in May 2013. Windows download requires the Visual C++ 2012 Redistributable which can be found here. Can be compiled for Linux and Mac; tested under Wine 1.7.25.

Source • Windows

GM Minesweeper (Easy)

I like Minesweeper so I tried to make my own. I only implemented the easy difficulty. Made using Game Maker; released April 2009 on the YoYo Games Sandbox for Windows. Works with Wine 1.7.25.

Source • Windows

Portal GM: Pre-Slice

Portal is my favourite video game so I tried to make a 2D version. There's no sound because I didn't know about SFXR back then. Made with GameMaker. Previously hosted on the YoYo Games Sandbox. Released May 2008. Only compiled for Windows, but works under Wine 1.7.25.

Source • Windows

SnakeMan

The result of combining certain elements of Snake and Pacman into one game. Made with GameMaker; released February 2008 on the YoYo Games Sandbox for Windows. Only compiled for Windows, but works under Wine 1.7.25.

Evolution is the Key (LD24)

You are trapped in a maze that constantly evolves around you. My first game, released in Java. Use WASD to move, MOUSE to look, and the LEFT MOUSE BUTTON to fire your weapon. Made in 72 hours for Ludum Dare 24. Scored #2 in Theme, #3 in Innovation.
